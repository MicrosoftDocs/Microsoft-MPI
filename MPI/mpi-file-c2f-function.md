# MPI\_File\_c2f function

Translates a C file handle to a Fortran file handle.

## Syntax

``` c++
MPI_Fint MPIAPI MPI_File_c2f(
   MPI_File file
);
```

## Parameters

  - *file*  
    C file handle.

## Return value

Fortran file handle.
