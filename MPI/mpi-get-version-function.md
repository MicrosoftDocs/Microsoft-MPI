# MPI\_Get\_version function

Returns the version number of MPI.

## Syntax

``` c++
int MPIAPI MPI_Get_version(
  _Out_ int *version,
  _Out_ int *subversion
);
```

## Parameters

  - *version* \[out\]  
    Version of MPI.

  - *subversion* \[out\]  
    Subversion of MPI.

## Return value

**MPI\_SUCCESS**

## Fortran

    MPI_GET_VERSION(VERSION, SUBVERSION, IERROR)
        INTEGER VERSION, SUBVERSION, IERROR
