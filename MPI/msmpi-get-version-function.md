# MSMPI\_Get\_version function

Returns the version number of the MSMPI feature set.

## Syntax

``` c++
int MPIAPI MSMPI_Get_version(void);
```

## Parameters

This function has no parameters.

## Return value

The version number of the MSMPI feature set.
