# MPI\_File\_f2c function

Translates a Fortran file handle to a C file handle.

## Syntax

``` c++
MPI_File MPIAPI MPI_File_f2c(
   MPI_Fint file
);
```

## Parameters

  - *file*  
    Fortran file handle.

## Return value

C file handle.
